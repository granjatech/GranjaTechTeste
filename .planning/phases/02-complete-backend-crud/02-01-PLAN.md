**get_all(pool: &PgPool, user_id: i32, user_role: &str) -> Result&lt;Vec&lt;LoteResponseDto&gt;, AppError&gt;**
- Role filtering: Admin = all lotes, Produtor = lotes where Granja.UsuarioId = user_id (JOIN Granjas), Financeiro = lotes where Granja.UsuarioId IN (SELECT ProdutorId FROM FinanceiroProdutor WHERE FinanceiroId = user_id)
- SQL must SELECT all Lote columns using PascalCase DB names with sqlx(rename) on model. Do NOT use SQL aliases -- let the Lote struct's sqlx(rename) handle mapping.
- After fetching Vec&lt;Lote&gt;, map each to LoteResponseDto using a private `fn map_lote_to_response(lote: &Lote) -> LoteResponseDto` function (per D-02).
- Computed properties in map function:
  - `idade_atual_dias = (Utc::now().date_naive() - lote.data_entrada.date_naive()).num_days() as i32`
  - `mortalidade_acumulada = lote.quantidade_aves_inicial - lote.quantidade_aves_atual`
  - `pct_mortalidade = if inicial > 0 { Decimal::from(mortalidade_acumulada) / Decimal::from(inicial) * Decimal::from(100) } else { Decimal::ZERO }`
  - `viabilidade = Decimal::from(100) - pct_mortalidade`
  - `densidade_atual = if area > Decimal::ZERO { Decimal::from(atual) / area } else { Decimal::ZERO }`

**get_by_id(pool, id, user_id, user_role) -> Result&lt;LoteResponseDto, AppError&gt;**
- Fetch Lote by ID, then verify access (same role-check pattern as GranjaService::get_by_id, checking via granja ownership)
- Fetch the Granja for the lote to check UsuarioId for access control
- Map to LoteResponseDto

**create(pool, dto: &CreateLoteDto, user_id, user_role, user_email) -> Result&lt;LoteResponseDto, AppError&gt;**
- Block Financeiro role
- Verify user has access to dto.granja_id (Admin=any, Produtor=own granja)
- Generate sequential code: `SELECT MAX("Id") FROM "Lotes"` then `format!("LT-{:03}", max_id + 1)`
- INSERT with QuantidadeAvesAtual = dto.quantidade_aves_inicial, Status = "Ativo", DataCriacao = Utc::now()
- Call `AuditoriaService::registrar_log(pool, user_id, user_email, "CRIACAO_LOTE", &format!("Lote '{}' criado na granja {}", codigo, granja_id))` (per D-03)
- Return mapped LoteResponseDto

**update(pool, id, dto: &UpdateLoteDto, user_id, user_role, user_email) -> Result&lt;LoteResponseDto, AppError&gt;**
- Block Financeiro role
- Fetch existing lote and verify access
- Delta logic for QuantidadeAvesInicial: `delta = dto.quantidade_aves_inicial - existing.quantidade_aves_inicial; new_atual = existing.quantidade_aves_atual + delta; if new_atual < 0 { new_atual = 0; }`
- UPDATE with RETURNING
- Audit log with "ATUALIZACAO_LOTE"

**delete(pool, id, user_id, user_role, user_email) -> Result&lt;(), AppError&gt;**
- Block Financeiro role
- Verify access, DELETE, audit log "DELECAO_LOTE"

**registrar_mortalidade(pool, lote_id, dto: &CreateRegistroMortalidadeDto, user_id, user_role, user_email) -> Result&lt;RegistroMortalidadeDto, AppError&gt;**
- Only Admin + Produtor (block Financeiro)
- Fetch lote, verify access
- Clamp quantity: `let qtd = dto.quantidade.min(lote.quantidade_aves_atual);`
- UPDATE Lotes SET QuantidadeAvesAtual = QuantidadeAvesAtual - qtd WHERE Id = lote_id
- INSERT INTO RegistrosMortalidade with ResponsavelRegistro = user_email, DataCriacao = Utc::now()
- Map to RegistroMortalidadeDto with computed: `idade_dias = (mortalidade.data.date_naive() - lote.data_entrada.date_naive()).num_days() as i32`, `percentual_mortalidade_dia = if inicial > 0 { Decimal::from(qtd) / Decimal::from(inicial) * Decimal::from(100) } else { Decimal::ZERO }`
- Audit log "REGISTRO_MORTALIDADE"

**listar_mortalidades(pool, lote_id, user_id, user_role) -> Result&lt;Vec&lt;RegistroMortalidadeDto&gt;, AppError&gt;**
- Verify access to lote
- Fetch all RegistroMortalidade for lote, ordered by Data DESC
- Map each with computed idade_dias and percentual_mortalidade_dia (need lote.data_entrada and lote.quantidade_aves_inicial)
