# Phase 4 — UI Design Contract

> Visual and interaction contract for Vue Scaffold + Auth. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none (Vuetify 3 is the design system) |
| Preset | not applicable |
| Component library | Vuetify 3 (v-components) |
| Icon library | @mdi/font (Material Design Icons — Vuetify default) |

Source: CONTEXT.md D-03, React theme.js fontFamily

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, chip internal padding |
| sm | 8px | Compact element spacing, inline padding |
| md | 16px | Default element spacing, drawer internal padding |
| lg | 24px | Section padding, card content padding, container py |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Login card internal padding (p: 6 = 48px from React) |
| 3xl | 64px | Page-level vertical spacing |

Exceptions:
- Navigation drawer width: 280px (locked by CONTEXT.md D-14)
- AppBar height: 64px desktop, 56px mobile (Vuetify default toolbar height)
- Navigation list item min-height: 48px (matches React 48px touch target)
- Login avatar: 80x80px (matches React LoginPage)

Source: React ResponsiveNavigation.js drawerWidth=280, LoginPage.js p:6, MUI spacing unit 8px

## Typography

All sizes are at fontScale=1.0. Font scale multiplier (0.85-1.3, step 0.1) applies to all sizes dynamically via CSS custom property `--font-scale` or Vuetify theme override.

| Role | Size | Weight | Line Height | Vuetify Mapping |
|------|------|--------|-------------|-----------------|
| Body | 16px (1rem) | 400 (regular) | 1.6 | body-1 / default text |
| Label | 14px (0.875rem) | 500 (medium) | 1.5 | body-2, button, nav items |
| Heading | 24px (1.5rem) | 600 (semibold) | 1.4 | h4 — page titles in PageContainer |
| Display | 40px (2.5rem) | 700 (bold) | 1.2 | h1 — not used in Phase 4 but declared for scale |

Additional sizes used in Phase 4:
- Caption: 12px (0.75rem), weight 400, line-height 1.4 — role chip labels, breadcrumb text
- Subtitle: 17px (1.0625rem), weight 400, line-height 1.5 — page subtitles in PageContainer
- Nav heading (drawer brand): 18px (1.125rem), weight 700, line-height 1.4 — "GranjaTech" in drawer

Font scale constants (from React AccessibilityContext):
- Default: 1.0
- Min: 0.85
- Max: 1.3
- Step: 0.1

Source: React theme.js createTypography, CONTEXT.md D-03

## Interaction States

### Login Form

| State | Visual |
|-------|--------|
| Default | Email field autofocused, submit button enabled only when both fields non-empty |
| Loading | Button shows circular progress (24px), fields disabled, button disabled |
| Error | `v-alert type="error"` appears above button with message "Email ou senha invalidos." |
| Success | Navigate to "/" (dashboard route), no visible success state |

### Navigation

| State | Visual |
|-------|--------|
| Desktop (md+) | Permanent drawer, app bar offset by 280px, no hamburger |
| Mobile (<md) | Temporary overlay drawer, hamburger icon in app bar, drawer closes on nav click |
| Active route | Nav item: primary background, white text and icon |
| Hover (inactive) | Nav item: action.hover background (rgba(0,0,0,0.04) light / rgba(255,255,255,0.08) dark) |
| Logout hover | Error-light background (#EF5350), white text |

### Accessibility Controls

| State | Visual |
|-------|--------|
| Dark mode off | Moon icon (mdi-weather-night), tooltip "Ativar tema escuro" |
| Dark mode on | Sun icon (mdi-weather-sunny), tooltip "Ativar tema claro" |
| Font at max (1.3) | Increase button disabled, opacity 0.5 |
| Font at min (0.85) | Decrease button disabled, opacity 0.5 |
| Reset | Returns to light mode + fontScale 1.0, removes localStorage key |

### Route Guards

| State | Visual |
|-------|--------|
| Unauthenticated visiting protected route | Redirect to /login, no flash of content |
| Authenticated visiting /login | Redirect to / |
| Token expired (401 from API) | Axios interceptor clears token, redirects to /login |

## Vuetify Theme Configuration Contract

The Vuetify plugin (`src/plugins/vuetify.ts`) must configure:

1. **Light theme** with all colors from the palette table above
2. **Dark theme** with dark-mode-specific background/text overrides
3. **Default theme** set from accessibility store (localStorage hydration)
4. **Global defaults**: `VBtn` rounded="lg", `VCard` rounded="lg", `VTextField` variant="outlined" rounded="lg", `VAppBar` flat + border
5. **Shape**: `borderRadius: 12` (default for cards/papers), buttons/inputs at 8px
6. **Transitions**: 200ms ease-in-out for color/background-color/border-color (global CSS)
7. **Scrollbar**: Custom thin scrollbar matching React (8px width, themed track/thumb)
8. **Focus-visible**: 2px solid outline (primary on light, #90CAF9 on dark), offset 2px
9. **Selection**: Background primary-dark on dark (#388E3C) / primary-light on light (#66BB6A), white text
