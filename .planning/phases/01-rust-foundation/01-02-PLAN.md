**CRITICAL -- PascalCase column mapping:** Use `#[sqlx(rename = "ColumnName")]` on each field per the RESEARCH.md recommendation. The EF Core schema uses quoted PascalCase identifiers (`"Id"`, `"Nome"`, `"SenhaHash"`). Rust fields must be snake_case. Example:

```rust
#[derive(Debug, Clone, Serialize, Deserialize, sqlx::FromRow)]
pub struct Granja {
    #[sqlx(rename = "Id")]
    pub id: i32,
    #[sqlx(rename = "Codigo")]
    pub codigo: String,
    #[sqlx(rename = "Nome")]
    pub nome: String,
    #[sqlx(rename = "Localizacao")]
    pub localizacao: Option<String>,
    #[sqlx(rename = "UsuarioId")]
    pub usuario_id: i32,
}
```

**Type mapping rules (from RESEARCH.md):**
- `int` -> `i32`
- `string` -> `String`
- `string?` -> `Option<String>`
- `DateTime` / `DateTime?` -> `DateTime<Utc>` / `Option<DateTime<Utc>>` (DB uses `timestamp with time zone`)
- `decimal` / `decimal?` -> `rust_decimal::Decimal` / `Option<rust_decimal::Decimal>`
- `bool` -> `bool`
- `double` / `double?` -> `f64` / `Option<f64>`

**DO NOT include navigation properties** (e.g., `pub usuario: Usuario` on Granja). SQLx has no lazy loading. Only include columns that exist in the database table.

**Part B: JWT Claims Extractor** -- Per D-05, create `middleware/jwt.rs`:

```rust
use actix_web::{FromRequest, HttpRequest, dev::Payload, web};
use jsonwebtoken::{decode, DecodingKey, Validation, Algorithm};
use serde::{Deserialize, Serialize};
use std::future::{Ready, ready};

use crate::config::Config;
use crate::errors::AppError;

#[derive(Debug, Serialize, Deserialize, Clone)]
pub struct Claims {
    pub nameid: String,   // user ID as string (matches .NET ClaimTypes.NameIdentifier)
    pub email: String,    // matches .NET ClaimTypes.Email
    pub role: String,     // matches .NET ClaimTypes.Role (profile name)
    pub exp: usize,
    pub iss: String,
    pub aud: String,
}

impl Claims {
    pub fn user_id(&self) -> Result<i32, AppError> {
        self.nameid.parse::<i32>()
            .map_err(|_| AppError::Unauthorized("Token invalido: nameid nao e um inteiro".into()))
    }
}

impl FromRequest for Claims {
    type Error = actix_web::Error;
    type Future = Ready<Result<Self, Self::Error>>;

    fn from_request(req: &HttpRequest, _payload: &mut Payload) -> Self::Future {
        let result = extract_claims(req);
        ready(result.map_err(actix_web::Error::from))
    }
}

fn extract_claims(req: &HttpRequest) -> Result<Claims, AppError> {
    let config = req.app_data::<web::Data<Config>>()
        .ok_or_else(|| AppError::Internal("Configuracao nao encontrada".into()))?;

    let auth_header = req.headers()
        .get("Authorization")
        .ok_or_else(|| AppError::Unauthorized("Token nao fornecido".into()))?
        .to_str()
        .map_err(|_| AppError::Unauthorized("Header Authorization invalido".into()))?;

    let token = auth_header
        .strip_prefix("Bearer ")
        .ok_or_else(|| AppError::Unauthorized("Formato do token invalido. Use: Bearer <token>".into()))?;

    let mut validation = Validation::new(Algorithm::HS256);
    validation.set_issuer(&[&config.jwt_issuer]);
    validation.set_audience(&[&config.jwt_audience]);

    let token_data = decode::<Claims>(
        token,
        &DecodingKey::from_secret(config.jwt_key.as_bytes()),
        &validation,
    )
    .map_err(|e| AppError::Unauthorized(format!("Token invalido: {}", e)))?;

    Ok(token_data.claims)
}
```
