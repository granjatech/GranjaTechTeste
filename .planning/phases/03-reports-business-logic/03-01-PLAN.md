- **Fully implemented methods** (per D-02, mirror .NET individual compute pattern):

  a) `pub async fn calcular_iep(pool: &PgPool, lote_id: i32) -> Result<Decimal, AppError>`:
     - Query lote from "Lotes" WHERE "Id" = $1
     - If lote not found, return Ok(Decimal::ZERO)
     - Query most recent pesagem: `SELECT "PesoMedioGramas" FROM "PesagensSemanais" WHERE "LoteId" = $1 ORDER BY "DataPesagem" DESC LIMIT 1`
     - If no pesagem, return Ok(Decimal::ZERO)
     - Calculate idade_dias = (Utc::now().date_naive() - lote.data_entrada.date_naive()).num_days()
     - If idade == 0, return Ok(Decimal::ZERO)
     - ganho_peso_kg = (peso_medio - 45) / 1000  (45g initial weight)
     - viabilidade = compute from lote (quantidade_aves_atual * 100 / quantidade_aves_inicial)
     - ca = Self::calcular_conversao_alimentar(pool, lote_id).await?
     - If ca == 0, return Ok(Decimal::ZERO)
     - iep = (ganho_peso_kg * viabilidade * 100) / (ca * idade)
     - Return iep.round_dp(2)

  b) `pub async fn calcular_conversao_alimentar(pool: &PgPool, lote_id: i32) -> Result<Decimal, AppError>`:
     - Query lote from "Lotes"
     - Query total racao: `SELECT COALESCE(SUM("QuantidadeKg"), 0) FROM "ConsumosRacao" WHERE "LoteId" = $1`
     - Query most recent pesagem (same as IEP)
     - If no pesagem or total_racao == 0, return Ok(Decimal::ZERO)
     - ganho_total_por_ave_gramas = peso_medio - 45
     - ganho_total_lote_kg = (ganho_total_por_ave_gramas * quantidade_aves_atual) / 1000
     - ca = if ganho > 0 { total_racao / ganho } else { 0 }
     - Return ca.round_dp(3)

  c) `pub async fn calcular_gmd(pool: &PgPool, lote_id: i32) -> Result<Decimal, AppError>`:
     - Query pesagens: `SELECT "PesoMedioGramas", "SemanaVida", "DataPesagem" FROM "PesagensSemanais" WHERE "LoteId" = $1 ORDER BY "SemanaVida" ASC`
     - Use a helper struct `#[derive(sqlx::FromRow)] struct PesagemRow { peso_medio_gramas: Decimal, semana_vida: i32, data_pesagem: DateTime<Utc> }` with appropriate `#[sqlx(rename = "...")]`
     - If count < 2, return Ok(Decimal::ZERO)
     - Iterate pairs: ganho_semanal = current.peso - previous.peso, gmd_semanal = ganho / 7
     - Average all gmd_semanal values, round to 2 decimal places

  d) `pub async fn verificar_alertas(pool: &PgPool, lote_id: i32) -> Result<Vec<AlertaParametroDto>, AppError>`:
     - Query lote
     - If not found, return Ok(vec![])
     - Compute percentual_mortalidade = if inicial > 0 { ((inicial - atual) as Decimal * 100) / inicial as Decimal } else { 0 }
     - If mortalidade > MORTALIDADE_MAXIMA: push AlertaParametroDto { tipo_alerta: "Mortalidade", severidade: if > 8 "Critica" else "Alta", valor_atual, valor_maximo: Some(MORTALIDADE_MAXIMA), unidade: "%", recomendacao: Some("Investigar causas e implementar medidas sanitarias") }
     - Compute densidade = if area > 0 { atual / area } else { 0 }; if > DENSIDADE_MAXIMA: push density alert (severidade: "Media")
     - Query recent air quality: `SELECT ... FROM "MedicoesQualidadeAr" WHERE "LoteId" = $1 AND "DataHora" >= $2 ORDER BY "DataHora" DESC LIMIT 5` (where $2 = Utc::now() - 1 day)
     - For each medicao: check NH3 > NH3_MAXIMO (severidade: if >35 "Critica" else "Alta"), check temperature <18 or >33 (severidade: if <15 or >35 "Critica" else "Media")
     - Return alertas vec

  e) `pub async fn comparar_com_industria(pool: &PgPool, lote_id: i32) -> Result<ComparacaoIndustriaDto, AppError>`:
     - Query lote (for identificador, idade, viabilidade)
     - If not found, return default ComparacaoIndustriaDto
     - Call calcular_ca, calcular_iep, calcular_gmd
     - Build 4 MetricaComparacaoDto (CA, GMD, Viabilidade, IEP) with:
       - Status logic: for CA (lower is better): <=excelencia="Excelente", <=padrao="Bom", else "Abaixo"
       - For GMD/Viabilidade/IEP (higher is better): >=excelencia="Excelente", >=padrao="Bom", else "Abaixo"
     - Note: .NET has 4 metrics in the scoring (CA, GMD, Viabilidade, IEP) but peso_medio is also a field on ComparacaoIndustriaDto. The .NET does NOT include peso_medio in the scoring. Set peso_medio to a default MetricaComparacaoDto with zeros.
     - pontuacao: Average of scores (Excelente=100, Bom=75, Abaixo=50) across 4 metrics, round to 1dp
     - classificacao: >=90 "Excelente", >=75 "Bom", >=60 "Regular", else "Ruim"
     - Helper: `fn calcular_percentual_diferenca(atual, padrao) -> Decimal` = if padrao==0 { 0 } else { ((atual-padrao)/padrao*100).round_dp(2) }
